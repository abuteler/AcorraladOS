AcorraladOS
===========

Mi segundo juego basado en un juego que me gustaba mucho de chico.
El nombre es una contracción de la palabra castellana "acorralados" con las siglas en inglés que se refieren a Operative System.
Y además termina en dos, xq es mi 2do juego. :)

My second game re-discovery project.
The name is a contraction between the spanish word for "cornered", meaning entrapment, and OS for Operative System.
Plus, the las 3 letters of the game's name are "dos" which means two, for the game being my second. :)
